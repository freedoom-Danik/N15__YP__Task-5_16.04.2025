# N15__YP__Task-5_16.04.2025
Задание 5.
1.	Создайте приложение Windows Forms в среде MS Visual Studio. 
2.	Поместите элемент MenuStrip в главную форму в окне проектирования. 

Контрольные вопросы 
1.	 Какие элементы управления используются для создания главного меню приложения? 
Для главного меню используется MenuStrip с ToolStripMenuItem.
2.	Какие элементы управления используются для создания контекстного меню элемента управления? 
Для контекстного меню используется ContextMenuStrip с ToolStripMenuItem.
3.	Как в режиме разработки указать элементу управления его контекстное меню? 
Контекстное меню назначается через свойство ContextMenuStrip в дизайнере или программно.
4.	Какие типы данных используются для создания пунктов меню? 
Для пунктов меню используются типы ToolStripMenuItem, ToolStripSeparator и другие производные от ToolStripItem.
5.	Поясните механизм синхронизации событий от разных элементов управления. 
Синхронизация событий осуществляется через общие обработчики событий, использование аргумента sender и привязку к командам.
